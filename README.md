# StreetVoice 資料擷取工具

一個基於 Node.js 的網頁工具，用於從 StreetVoice 平台自動擷取歌詞和音樂人資訊並整合到 Excel 檔案中。

## 功能特色

- 📋 **Excel 檔案處理**: 支援上傳和解析 .xlsx 格式檔案
- 🎵 **歌詞擷取**: 自動從 StreetVoice 歌曲頁面擷取完整歌詞
- 👤 **音樂人資訊**: 擷取音樂人類別和詳細介紹
- 🎛️ **彈性選項**: 可自由選擇需要擷取的資料類型
- 📊 **即時進度**: 顯示處理進度和狀態
- 💾 **自動輸出**: 處理完成後自動生成下載檔案

## 系統需求

- Node.js 16.0 或更高版本
- npm 或 yarn
- 網路連線（無需安裝瀏覽器）

## 安裝步驟

1. **克隆或下載專案**
   ```bash
   cd streetvoice-scraper
   ```

2. **安裝依賴套件**
   ```bash
   npm install
   ```

3. **啟動服務器**
   ```bash
   npm start
   ```

4. **開啟瀏覽器**
   訪問 `http://localhost:3000` 使用工具

## 使用方法

### Excel 檔案格式要求

上傳的 Excel 檔案必須包含以下欄位：
- `作品名稱`: 歌曲名稱，需包含 StreetVoice 歌曲頁面的超連結
- `作者`: 音樂人名稱，需包含 StreetVoice 音樂人頁面的超連結

### 操作步驟

1. **上傳檔案**: 選擇或拖拽 .xlsx 格式的 Excel 檔案
2. **選擇選項**: 勾選需要擷取的資料類型
   - ✅ 歌詞：完整歌詞內容
   - ✅ 音樂人類別：樂團、歌手等分類
   - ✅ 音樂人介紹：完整介紹和社群連結
3. **開始處理**: 點擊「開始處理」按鈕
4. **下載結果**: 處理完成後下載新的 Excel 檔案

### 輸出格式

處理後的 Excel 檔案會在原有欄位後新增：
- `歌詞`: 完整歌詞內容（如有選擇）
- `音樂人類別`: 音樂人分類標籤（如有選擇）
- `音樂人介紹`: 音樂人詳細介紹（如有選擇）

## 技術架構

### 後端技術
- **Express.js**: Web 服務器框架
- **Axios**: HTTP 客戶端請求
- **Cheerio**: 服務端 DOM 解析
- **ExcelJS**: Excel 檔案讀寫
- **Multer**: 檔案上傳處理

### 前端技術
- **HTML5**: 結構化網頁
- **CSS3**: 響應式樣式設計
- **JavaScript**: 互動功能和 AJAX

### 資料擷取邏輯
- 使用 HTTP 請求直接獲取網頁內容
- 使用 Cheerio 解析 HTML 結構
- 多重策略智能提取歌詞和音樂人資訊
- 並行處理提升效率
- 錯誤處理和重試機制

```
streetvoice-scraper/
```


- `PORT`: 服務器端口 (預設: 3000)

## 技術優勢

✅ **無需瀏覽器依賴**: 使用純 HTTP 請求，無需安裝 Chrome 或 Puppeteer
✅ **更快的速度**: HTTP 請求比瀏覽器自動化快 3-5 倍
✅ **更少資源消耗**: 記憶體使用量降低約 80%
✅ **更高穩定性**: 避免瀏覽器崩潰和超時問題
✅ **並行處理**: 支援多重並行請求提升效率

## 注意事項

- 處理大量資料時請耐心等待，避免頻繁請求被網站封鎖
- 建議分批處理大型檔案，每次不超過 100 筆資料
- 網絡不穩定時可能出現擷取失敗，已標記為「擷取失敗」
- 檔案處理完成後會自動清理暫存檔案
